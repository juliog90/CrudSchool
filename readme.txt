Guia rapida para probar este programa.

1. Importar PersonasDB.sql a MySql con el manejador que se tenga del mismo
2. Cargar el proyecto de Java en IntelliJ
3. Importar las dependencias de Maven: Una ventana saldra con el mensaje de import o auto import
   Escoger una de las dos
4. Click derecho en src->main->java->Main y escoger run Main.main()
